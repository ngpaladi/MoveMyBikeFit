About

MoveMyBikeFit is an open-source gravel bike geometry comparison and fit tool. Select any two bikes to overlay their silhouettes, enter your fit measurements to find stem and saddle combinations, and compare frame geometry side by side.

Adding a bike

  1. Fork the repository on GitHub.
  2. Use Add Bike above to generate a YAML file from official geometry data.
  3. Drop the file in _data/bikes/ and open a pull request. Include the source URL in the PR description.
