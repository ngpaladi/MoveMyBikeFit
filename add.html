    Submitting to MoveMyBikeFit
    
    
      
  1. Fork the MoveMyBikeFit repository on GitHub.
    2. 
      
  2. Add your downloaded YAML file to _data/bikes/
    3. 
      
  3. Name the file: {brand}-{model}-{year}.yml (all lowercase, spaces as hyphens)
    4. 
      
  4. Open a Pull Request. Include the source URL for the geometry data in your PR description.
    5. 
    

    

      Geometry data should come from the official manufacturer website or press kit. Please double-check values before submitting.
